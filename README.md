# Typora Themes

A small collection of [Typora](https://typora.io) themes.

| Theme | Style |
|---|---|
| [**Catppuccin**](#catppuccin) | Soothing pastel — all 4 official flavors (Latte · Frappé · Macchiato · Mocha) |
| [**Plannotator**](#plannotator) | Dark deep-navy with a lavender / orange accent |

## Install (any theme)

1. In Typora: **Preferences → Appearance → Open Theme Folder**.
2. Copy the `.css` file(s) you want into that folder.
3. Restart Typora and pick the theme from the **Themes** menu.

> Requires Typora ≥ 0.9.12 (Windows) / 0.9.9.5.1 (macOS).

## Catppuccin

A faithful port of [Catppuccin](https://catppuccin.com) — the community-driven
soothing pastel theme — to Typora. All four official flavors are included as
separate, self-contained themes, built from the official
[`catppuccin/palette`](https://github.com/catppuccin/palette) **v1.8.0** hex
values.

> Tip: drop in only the flavors you want — each file is standalone. Mocha is
> the flagship dark flavor; Latte is the light one.

### Features

- **Style-guide-faithful syntax highlighting** — CodeMirror tokens (both fenced
  code blocks and source mode) follow the official
  [Catppuccin Style Guide](https://github.com/catppuccin/catppuccin/blob/main/docs/style-guide.md):
  keywords = Mauve, strings = Green, functions = Blue, numbers/constants = Peach,
  types = Yellow, operators = Sky, comments = Overlay (italic).
- **Headings** use a calm accent sweep: h1 Mauve → h2 Blue → h3 Sapphire →
  h4 Teal → h5 Green → h6 Subtext.
- Full coverage of links, blockquotes, tables, task lists, YAML front matter,
  math, diagrams, TOC, focus mode, GitHub-style alert callouts, sidebar /
  file-list, search highlights, and a light print stylesheet.
- **Inter** for body, **JetBrains Mono** for code (system fallbacks).

### Semantic mapping

| Markdown / UI element | Catppuccin color |
|---|---|
| Body text | Text |
| Links | Blue (hover: Sky) |
| **Bold** | Peach |
| *Italic* | Mauve |
| `inline code` | Green on Surface0 |
| List markers | Peach |
| Blockquote bar | Mauve → Blue → Teal (by depth) |
| Headings h1–h6 | Mauve · Blue · Sapphire · Teal · Green · Subtext0 |
| Selection / search | Mauve (active match: Peach) |
| Page / editor bg | Base · Mantle (sidebar) · Surface0 (panels) |
| Borders | Surface0 / Surface1 |

### Regenerating

All four flavors share one rule set and differ only in their 26-color palette.
They are generated by [`build-catppuccin.py`](./build-catppuccin.py) — edit the
template or palette once and rerun to keep every flavor in sync:

```bash
python3 build-catppuccin.py
```

## Plannotator

A dark Typora theme ported from the **Plannotator** color scheme used in
[backnotprop/plannotator](https://github.com/backnotprop/plannotator) — the
local, browser-based review surface for AI coding agents.

The palette comes from [`plannotator-zed.json`](./plannotator-zed.json) (the
Plannotator Zed editor theme): deep navy surfaces with a lavender primary accent
(`#9a9dff`) and a warm orange secondary accent (`#f47700`).

![palette](https://img.shields.io/badge/background-%23070b14-070b14) ![palette](https://img.shields.io/badge/accent-%239a9dff-9a9dff) ![palette](https://img.shields.io/badge/orange-%23f47700-f47700) ![palette](https://img.shields.io/badge/green-%233fc168-3fc168)

### Features

- **Faithful syntax highlighting** — CodeMirror tokens in both fenced code
  blocks (`.cm-s-inner`) **and** source code mode (`.cm-s-typora-default`) are
  mapped from the Zed `syntax` palette.
- **Headings** each get their own color — a cool→warm gradient:
  h1 indigo → h2 cyan → h3 teal → h4 green → h5 amber → h6 muted.
- Full coverage of blockquotes, tables, task lists, YAML front matter, math,
  diagrams, TOC, focus mode, callouts, sidebar/file-list, search highlights,
  and print styles.
- **Inter** for body, **JetBrains Mono** for code (falls back to system fonts).
